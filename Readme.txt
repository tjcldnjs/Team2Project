CrazyAcade
Team2Project 


프로젝트 소개
커스텀 맵을 사용할 수 있는 CrazyAcade를 Swing으로 구현한 게임입니다.

개발 기간
24.05.10 - 24.05,20

사용법
1P : 상하좌우(방향키), 물풍선(Spacebar)
2P : 상하좌우(AWSD), 물풍선(Shift)

게임 가이드
1~3번의 숫자를 콘솔창에 입력하면 미리 만들어둔 커스텀 맵을 즐길수 있다.
0번을 입력하면 자신이 직접 맵을 커스텀 할 수도 있으며, 한 번 클릭하면 빨강(부숴지지 않는 블럭), 두 번 클릭하면 파랑(부숴지는 블럭), 세번 클릭하면 흰색(블럭을 만들지 않음)을 찍어 직접 맵을 만들어 즐길 수 있다.
기본 셋팅된 블럭은 지울 수 없다.

물풍선의 사용횟수는 제한이 없으며 물풍선을 이용하여 상대를 물방울에 가두면 이기는 게임이다.
자신이 놓은 물풍선에도 죽을 수 있다.
